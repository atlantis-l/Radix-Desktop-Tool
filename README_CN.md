## 运行程序

```shell
# 第一步
npm install
# 第二步
npm run dev
```

## 构建应用

```shell
# Windows
npm run build-win
# MacOS
npm run build-mac
# Linux
npm run build-linux
# 生成的可执行文件存放在release目录
```
